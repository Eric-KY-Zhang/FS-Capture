# FS Capture 使用说明

FS Capture 用于一键抓取 A 股、港股、美股、韩股上市公司的年报/季报文件和基础财务数据，并生成 Excel 底稿。

## 日常使用

1. 双击根目录的 `FS Capture.exe`。
2. 勾选交易所，添加股票代码，点击“确认”识别公司名称。
3. 选择年份区间和报告类型。
4. 点击“开始抓取”。
5. 结果会保存在 `output/` 文件夹，Excel 底稿和下载报告都会直接平铺在这一层，不再按公司或市场继续嵌套。

## 根目录文件

- `FS Capture.exe`：主程序，日常只需要双击它。
- `_internal/`：程序运行依赖文件夹，请不要删除或移动。
- `config.toml`：本机配置文件，默认已隐藏；可在程序内“设置”里修改。
- `config.example.toml`：配置模板，默认已隐藏，不含个人密钥。
- `output/`：抓取结果，Excel 底稿和报告文件都平铺在这里。
- `data/`：缓存和日志。
- `development/`：源码、构建脚本、测试和开发文档，默认已隐藏，普通使用不用打开。

## 韩股说明

韩股官方披露和财务数据主要来自 DART OpenAPI。若无法注册 DART，可先不勾选韩股；A 股、港股、美股功能不受影响。

可选替代路径：
- 直接在 DART 网站手工检索公司披露文件，适合作为少量公司兜底。
- KRX/KIND 可用于交易所公告检索，但财务报表字段和文档下载自动化不如 DART 稳定。
- 公司 IR 官网通常能找到英文/韩文年报，但不同公司页面结构差异大，适合人工补充，不适合做统一自动抓取主数据源。

## 开发

源码和构建脚本在 `development/`：

```bat
cd development
run.bat
build.bat
```
