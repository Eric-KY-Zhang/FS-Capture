# 上市公司财务数据查询

Excel + VBA 财务数据抓取工具,用于把多家上市公司的资产负债表、利润表、现金流量表和标准财务指标整理成同业对标宽表。

## 快速开始

### 1. 前置条件

- Windows + Excel,推荐 Microsoft 365 / Excel 2016+
- Python 3.x + `openpyxl` + `pywin32`
- Excel 需要启用「信任对 VBA 工程对象模型的访问」:
  Excel → 文件 → 选项 → 信任中心 → 信任中心设置 → 宏设置 → 勾选最后一项

### 2. 生成空模板

```powershell
py tools/build_template.py
```

输出:

```text
上市公司财务数据查询.xlsx
```

### 3. 安装 VBA 模块并生成 xlsm

```powershell
py tools/install_modules.py
```

脚本会:

- 打开 `上市公司财务数据查询.xlsx` 或现有 `上市公司财务数据查询.xlsm`
- 注入 `modules/*.bas`
- 创建/刷新 A股、美股、港股报表 sheet 和 `美股_抓取诊断` / `港股_抓取诊断`
- 刷新「使用说明」
- 在「样本池」安装一键按钮
- 保存为 `上市公司财务数据查询.xlsm`

如果目录里只有旧版 `新浪财经行业数据查询V3.xlsm`,安装脚本会自动读取旧文件并另存为新文件名。

## 使用方式

打开 `上市公司财务数据查询.xlsm`,进入「样本池」:

- `A2`: 年度,例如 `2025`;留空表示取最新可用期间
- `A4`: 季度,可选 `全部 / Q1 / Q2 / Q3 / Q4`
- `B5`: 雪球 `xq_a_token` cookie,用于部分美股 20-F/中概股 fallback 和港股抓数
- 第 8 行起录入公司:
  - `A列`: 股票代码
  - `B列`: 股票简称
  - `C列`: 市场,可填 `A / US / HK / KR`;留空会自动判断 A/US/HK,KR 请手填

当前实际抓数支持 A股、美股和港股。韩股的市场位已预留,抓数逻辑后续接入。

## 输出表

Tab 顺序:

1. 使用说明
2. 样本池
3. A股_资产负债表
4. A股_利润表
5. A股_现金流量表
6. A股_指标表
7. 美股_资产负债表
8. 美股_利润表
9. 美股_现金流量表
10. 美股_指标表
11. 美股_抓取诊断
12. 港股_资产负债表
13. 港股_利润表
14. 港股_现金流量表
15. 港股_指标表
16. 港股_抓取诊断

宽表结构:

- 第 1 行: 公司名和代码
- 第 2 行: 报告期
- A/B列: 大类或指标类型、指标名称
- 指标表额外包含英文指标名列
- A股按报告期并集对齐,便于同行横向比较
- 美股/港股按每家公司自身可用期间展开,避免为其他公司的期间保留空列
- 港股单位为百万(各家公司报告币种,见 `港股_抓取诊断` 的 Unit 列),不做汇率换算

指标表统一只保留 18 个标准指标,A股、美股和港股共用同一套口径。

## 注意事项

- 美股 EDGAR 字段存在公司差异,诊断会写入 `美股_抓取诊断`。
- 港股来自雪球 HK API,诊断会写入 `港股_抓取诊断`。
- fuzzy 匹配只输出诊断推荐,不会自动写入正式财报。
- 雪球 cookie 过期时,需要重新复制 `xq_a_token` 到「样本池」`B5`。
- 韩股属于下一阶段扩展范围,当前 README 和工作簿已按新项目名预留。
